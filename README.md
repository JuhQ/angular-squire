# angular-squire
angularjs directive for the [squire rich text editor](https://github.com/neilj/Squire). 

Check out the [DEMO](http://hourlynerd.github.io/angular-squire/)


# depends on

```js
"angular": "~1.3.8",
"angular-strap": "~2.1.6",
"underscore": "~1.7.0",
"jquery": ">= 1.9.0",
"font-awesome": "~4.3.0",
"squire-rte": "~1.0.1"
```

# building

```bash
npm install bower -g
npm install gulp -g
npm install
bower install
gulp build
```
